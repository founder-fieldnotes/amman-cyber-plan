# Amman Cybersecurity & Digital Forensics Program

This repository contains the master blueprint for the Amman Cybersecurity & Digital Forensics Program — a 16-week dual-track training model designed to prepare learners for applied roles in cybersecurity and digital forensics.

## Program Snapshot

- 100 total learners
- 75 learners in Cybersecurity
- 25 learners in Digital Forensics
- Amman, Jordan
- Shared foundation in the opening phase, followed by track specialization and a final capstone phase

## Program Structure

The program is organized in three parts:

1. Shared Foundation  
   Learners build common baseline capability in technical workflow, systems thinking, documentation, and professional execution.

2. Track Specialization  
   Learners move into either the Cybersecurity or Digital Forensics track for applied, role-aligned technical training.

3. Capstone Phase  
   Learners bring their work together in a final applied project phase that emphasizes execution, communication, and readiness for real-world environments.
